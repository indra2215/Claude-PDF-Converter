Claude PDF Converter

PDF → Markdown for Claude
Active on claude.ai
How to use
1
Paste a PDF (Ctrl+V) or upload via Claude's file button
2
A toast appears — click "Convert to .md"
3
Click "Attach .md to Claude" — Markdown is attached as a .md file to your chat
4
Image PDFs are auto-detected and OCR'd via Tesseract
⚠ Accuracy Caveats
Image-based / scanned PDFs: OCR accuracy varies by scan quality
Handwritten text: unreliable — Tesseract is not trained for handwriting
Tables & complex layouts may flatten or merge incorrectly
Always verify converted content before relying on it
🔒 All processing is local · No data sent anywhere
